# Видеоконференции

Платформа для видеозвонков с возможностью автоматического конспектирования в реальном времени.

## Описание

Конференции будут проводится как обычно, но вместе с пользователями в конфе будет сидеть AI. Пользователи могут одновременно подгружать в контекст документы. AI будет вести динамическое конспектирование, выделяя ключевые слова (возможно также поясняя через интеренет источники).

Платформа будет иметь онлайн-доску (функционал доски хорошо скейлиться), которая также будет частью контекста.

Платформа будет иметь чат-бота / голосового помошника, который во время проведения конференции сможет давать контекстный ответ на основе содержания конспекта коныф и текущего состояния онлайн-доски.
